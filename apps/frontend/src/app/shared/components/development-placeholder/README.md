# app-development-placeholder

Pagina de estado para funcionalidades en desarrollo. Muestra un mensaje centrado con icono de casco y boton para volver atras.

## Uso

Para rutas que aun no estan implementadas pero no deben dar error 404.

```html
<app-development-placeholder></app-development-placeholder>
```

## Importante

- Navega hacia atras con `window.history.back()`, no con Router.
- No tiene inputs configurables.
- Texto en espanol hardcodeado.
- Usa clases CSS de variables de tema (color-primary, etc.).
